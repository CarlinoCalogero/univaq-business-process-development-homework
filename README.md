# Business Process Development - Homework
This repository contains the homework assignments for the **Business Process Development** course. The project focuses on modeling, analyzing, and implementing business processes using industry-standard notations and tools.

## Project Overview
The goal of this homework is to design a business process. The project covers the full lifecycle of a business process, from conceptual modeling to automated execution.

## Repository Structure
* `/models`: Contains `.bpmn` and `.dmn` files.
* `/src`: Implementation code for Service Tasks and Workers.
* `/docs`: Project documentation and process descriptions.
* `/resources`: Assets and configuration files.
